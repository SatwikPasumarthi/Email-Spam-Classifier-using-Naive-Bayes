# Spam-Classifier-using-Naive-Bayes
### Introduction
This Email Spam Classifier detects the unsolicited, unwanted, and virus-infested emails (called spam emails) using Naive bayes algorithm.

### Purpose of the Project
This project aims to classify the emails and gives us the output whether it is a SPAM emaail or HAM email.

### Dataset
[Link for the dataset](https://www.kaggle.com/venky73/spam-mails-dataset?select=spam_ham_dataset.csv)
